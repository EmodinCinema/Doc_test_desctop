Чтобы скачать библиотеку Electron нужно зайти в cmd. С помощью команды 'cd "путь к проекту"' и выполнить команду 'npm install'. После этого всё загрузиться. Для запуска программы нужно набрать 'npm start'

Список команд:(по порядку)
-cd "путь к папке"
-npm install (Скачитвание библиотеки Electron)
-npm start (Запуск проекта)
